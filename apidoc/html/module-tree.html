Module Hierarchy

  • testlink
    • testlink.api: Raw API Wrapper for Testlink
    • testlink.objects
    • testlink.server: Testlink conform automation server. Since version 1.9.2, Testlink supports the invocation of remote methods from the execution view. For make this work, following configurations must be made in Testlink:
      • Activate API in config.inc.php: $tlCfg->api->enabled = TRUE
      • Enabled XML-RPC calls from config.inc.php: $tlCfg->exec_cfg->enable_test_automation = ENABLED
      • Define a special custom field e.g. on TestCase level: RE-XMLRPX_url_tcase
      • Set the value of the field to <ADDR>:<PORT><PATH>, e.g.: 127.0.0.1:8000/RPC2
